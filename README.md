# Grocery Store Management System

OOP Course Project - Week 2

A Java-based grocery store management system implementing core OOP principles: classes, objects, encapsulation, and methods.

## Project Structure

src/
├── Main.java      - Main program demonstrating all features
├── Product.java   - Grocery product management
├── Customer.java  - Customer and membership system
└── Sale.java      - Sales transaction handling


## Features

### Product Class
- Track product inventory (ID, name, price, stock)
- Check stock availability
- Restock products
- Apply discounts
- Sell items with stock validation

### Customer Class
- Customer management with membership levels
- VIP status based on purchase history
- Automatic membership upgrades (Standard → Silver → Gold → Platinum)
- Discount system based on membership tier

### Sale Class
- Process sales transactions
- Add items to cart
- Apply customer discounts
- Calculate tax (12%)
- Track large sales

## How to Run

1. Clone the repository:

git clone https://github.com/W1theri/GroceryStore-OOP-Project.git


2. Open in IntelliJ IDEA

3. Run `Main.java`

- 3 core classes with 4+ private fields each
- Constructors (parameterized + default)
